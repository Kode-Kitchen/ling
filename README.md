# Ling

> REST Client built with [Electron](https://github.com/electron/electron), using [Bozon](https://github.com/railsware/bozon)

## Install bozon

```
$ npm install -g bozon
```

## Install dependencies

```
$ npm install
```

### Run

```
$ bozon start
```

### Package

```
$ bozon package
```
